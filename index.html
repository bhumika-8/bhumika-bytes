Projects...

  • Astrolite

    Astrolite is a space-themed platform featuring a Satellite Tracker that tracks live satellite positions using Satellite.js. It also showcases detailed Solar System data with engaging CSS animations. The site uses the Intersection Observer API for optimized performance, triggering animations only when elements are in view during scrolling, offering an interactive and smooth user experience.

  • Finacoaster

    A finance web application developed during a 24-hour hackathon, where users can earn and spend "Finzos," the in-game currency. User authentication is handled with Node.js and Express.js. The dashboard visualizes users' earnings and spending patterns, while a leaderboard displays top earners. REST APIs fetch live financial news to keep users informed.
